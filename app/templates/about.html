About Auto-Worksheet


Auto-Worksheet is a portfolio project built in Python using the Flask framework.

Initial article scraping is done using Selenium
Selenium is able to scrape dynamically loaded content, at the cost of speed.
The remaining scraping is done with BS4.
The worksheet is assembled with Python-DOCX.
